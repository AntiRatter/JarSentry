# JarSentry

JarSentry is a defensive Java tool designed to analyze Minecraft mod JAR files
for suspicious behavior using heuristic and trust-based analysis.

## Features
- Risk scoring (Low / Medium / High)
- Modrinth & Forge trust verification
- Obfuscation detection
- Known RAT fingerprint detection

## Disclaimer
This tool is for educational and defensive research purposes only.
It does not remove malware or exploit software.

## Build
Requires Java 17+
